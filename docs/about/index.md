# About the teams

投資堆疊 (Investment Stack) 是一個概念，指的是用於管理投資的分層架構或方法。這個詞語來自軟體開發領域的「技術堆疊」概念，延伸到投資領域，用來描述投資者用於做出決策和管理資金的工具、策略與資源的組合.
